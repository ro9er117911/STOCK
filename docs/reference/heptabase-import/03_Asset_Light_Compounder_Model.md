# Roper's Asset-Light Compounder Model

- **Strategy:** Build a portfolio of "toll-bridge" businesses—high-moat, niche technology or service companies with recurring revenue.
- **Criteria for Acquisitions:**
    1. High margins (EBITDA).
    2. High free cash flow conversion.
    3. Low capital intensity (Asset-light).
    4. Market leaders in small, overlooked niches.
